Spisegram — это самостоятельная C#-реализация серверного API Telegram, предназначенная для частных развертываний и расширения.

## Поддерживаемые функции

### Функции с открытым исходным кодом
- Уровень API: `224`
- Транспорты MTProto: `Abridged`, `Intermediate`
- Приватный чат
- Супергрупповой чат
- Каналы
- Поддержка ботов
- Стикеры
- Реакции
- Звёздные подарки
- Темы форума
- Бизнес функционал
- Личные сообщения
- Рассказы
- Чат-лист
- Поддержка ботов
- Сквозно-конечный шифрованный чат
- Настройки конфиденциальности и 2FA
- Автоматическое удаление сообщений
- Запланированные сообщения

### В скором времени...
- Голосовые и видеозвонки
- Темы и обои
- Вход через ключ доступа
- Вход по электронной почте
- Отправитель электронной почты
- Push-сервер (Firebase)
